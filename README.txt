Obsah tohoto pametoveho media:

src/ - vsechny soubory se zdrojovym kodem
  src/android_app/ - adresar obsahujici veskere soubory tykajici se androidove aplikace
  src/scripts/ - ruznorode skripty - tvorba grafu, prevod keras -> tensorflow ...
  src/clouds_recognizer/ - zdrojove kody k vytvoreni neuronove site pro detekci fotografie oblak
  src/clouds_classifer/ - zdrojove kody k vytvoreni neuronove site pro klasifikaci oblak
  src/dataset_download/ - zdrojove kody pro stahovani datasetu (ruzne verze) 
doc/ - zdrojove kody technicke zpravy vcetne jeji PDF verze
bin/ - prelozena aplikace pro Android

Instalace:

.apk soubor ze slozky bin/ preneseme na vybrany telefon s operacnim systemem Android. Aplikaci pote nainstalujeme otevrenim souboru.
